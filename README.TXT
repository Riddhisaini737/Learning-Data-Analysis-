⁕ Project 

Data Analysis of Weather and Air Quality Index (AQI) for the Year 2025 in New Delhi, India.


⁕ Objective

The objective of this project is to analyze monthly temperature and air quality (AQI) data for the year 2025.
This project aims to learn and practice basic concepts of data analysis and data cleaning using simple tools.


⁕ Tools Used

● Microsoft Excel: For entering, cleaning, analysing and visualising the data.

● Java: For creating and handling CSV files.

● Notepad: For creating the README File.



⁕ Analysis Performed

There are two CSV Files made:


● Temperature.csv

This file contains:

Minimum temperature of each month.

Maximum temperature of each month.

Average temperature of each month.

Climate remark describing the overall weather condition of the month.


● AQI.csv

This file contains:

Minimum AQI of each month.

Maximum AQI of each month.

Average AQI of each month.

Air quality remark describing the overall air quality.



⁕ Observations

● The overall average temperature in the year 2025 was 25°C.

● The hottest month recorded in 2025 was July with an average temperature of 33.1°C.

● The coldest month recorded in 2025 was January with an average of 14.6°C.


● The overall average AQI in the year 2025 was 155 falling under the unhealthy air quality category.

● The cleanest month recorded in 2025 was July, Average AQI- 78.

● The most polluted month recorded im 2025 was November, Average AQI- 359.
